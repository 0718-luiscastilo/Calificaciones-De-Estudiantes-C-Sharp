Sistema de Registro de Calificaciones en C#
Descripción
Este proyecto es una aplicación de consola desarrollada en C# que permite registrar las calificaciones de estudiantes y generar un análisis básico del grupo.

El sistema solicita datos al usuario y calcula automáticamente:

Promedio general
Número de estudiantes aprobados
Número de estudiantes reprobados

Funcionalidades
Registro de múltiples estudiantes
Almacenamiento de nombres y calificaciones en arreglos
Cálculo del promedio general
Clasificación de estudiantes (aprobados/reprobados)
Visualización de resultados en consola

Tecnologías utilizadas
C#
.NET (Aplicación de consola)
